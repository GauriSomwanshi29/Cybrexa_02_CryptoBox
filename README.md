## 📖 Overview

CryptoBox is a modern cybersecurity-focused web application designed to help users understand password security and generate strong passwords for safer online accounts.

The application provides real-time password analysis, strength evaluation, entropy estimation, breach detection using common password datasets, and a customizable password generator.

Built using modern frontend technologies, CryptoBox combines cybersecurity concepts with an intuitive and professional user interface.

## 📊 Entropy Analysis

CryptoBox estimates password entropy to measure randomness and predict resistance against brute-force attacks.

### Example

| Password Type   | Entropy  |
| --------------- | -------- |
| Weak Password   | Low      |
| Medium Password | Moderate |
| Strong Password | High     |

Higher entropy generally means better security.

---

## ⏳ Time-To-Crack Estimation

CryptoBox provides an estimated time required to crack a password using brute-force techniques.

### Example Outputs

⚠️ Few Seconds

⌛ Few Minutes

🕒 Few Hours

📅 Few Months

🏛️ Hundreds of Years
